==================
Execution Profiler
==================

The execution profiler tool outputs various metrics about the execution, like the number of executed instructions,
page faults, TLB misses, cache profile, etc.

Examples
--------

Required Plugins
----------------

* ``ExecutionTracer``
* ``CacheSim``
* ``InstructionCounter``

Optional Plugins
----------------

* ``ModuleTracer`` (for debug information)
